# Youth Leadership Hub — Operational Playbook
> Challenge 2.3 · Strategic Consulting · Open-source replication kit

A free, fully documented playbook for setting up a localised Youth Leadership Hub in a semi-urban Indian town — designed to run with zero paid full-time staff.

##  Live site
Deploy `index.html` via GitHub Pages — go to **Settings → Pages → Source: main branch** and your site will be live at `yourusername.github.io/youth-hub-playbook`.

---

## What's inside

| Section | What it covers |
|---|---|
| **Theory of change** | Outcomes defined backwards from civic impact, using Pravah's "5th Space" framing. Design principles from 5 existing models. |
| **Hub model** | How the hub runs without a paid coordinator. The 90-day member journey. Gender inclusion non-negotiables. |
| **Governance** | 5 rotating volunteer roles, decision-making protocols, accountability mechanisms. |
| **12-month calendar** | Full activity calendar across 2 cohorts — civic dialogues, skill sprints, field visits, public events. |
| **Funding model** | Minimum viable ₹3.6L/year budget across 5 diversified sources. No single funder above 40%. |
| **Pilot: Belagavi** | Feasibility analysis for Belagavi, Karnataka — 6-factor assessment + 12-month forecast. |
| **Replication checklist** | 20 actionable steps across 4 phases any volunteer team can follow to launch their own hub. |

## Key design principles

- **Zero paid staff** — the hub is fully volunteer-run after Month 10
- **Politically neutral** — charter-mandated; facilitators trained in facilitation-not-persuasion
- **Gender-inclusive by design** — 50% women mandatory, not aspirational
- **Self-sustaining loop** — Cohort 1 alumni become Cohort 2 facilitators
